• Install the Angular CLI if not done already
    npm install -g @angular/cli
  • Create a new project and its skeleton
    ng new my-app
    Patience, please. It takes time to set up a new project; most of it is spent installing npm packages
  • Launch the app
    cd my-app
    ng serve –open
    The ng serve command launches the server, watches your files, and rebuilds the app as you make changes to those files
    --open automatically opens the browser @ http://localhost:4200/
