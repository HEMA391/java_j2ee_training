  • You should see the spy class names as Untouched, Touched, Changed, Invalid
    The control has been visited: ng-touched and ng-untouched
    The control's value has changed: ng-dirty and ng-pristine
    The control's value is valid: ng-valid ng-invalid
  • Note that required is added in the input types
    If required fields are not filled, you should see corresponding messages from the Angular framework itself
  • Note that hidden is used for displaying form submission results
