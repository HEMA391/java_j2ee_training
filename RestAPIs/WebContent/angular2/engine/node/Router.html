Router Sample

  • Router Module
  • Navigate to the directory D:\ws\angular2
  • Create a new project and its skeleton

    • ng new router
  • Download all files from the Git
  • Navigate to D:\ws\angular2\router\src\app directory

    • Replace the contents of this directory with the files downloaded
  • Launch the app

    • cd router
    ng serve –open
  • If you get error like Cannot find module '@angular-devkit/core', navigate to D:\ws\angular\router
    Update npm: npm update -g @angular/cli
    npm i -D @angular-devkit/core
